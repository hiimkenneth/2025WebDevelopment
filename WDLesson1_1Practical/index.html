Web Technologies

HTML

HTML, which stands for Hypertext Markup Language, is the standard markup language for creating web pages. HTML describes the strucure of a web page, consists of a series of elements, HTML elements tell the browser how to display this content. They also label pieces, such as "this is a heading, link, paragraph", etc.

CSS

CSS, which stands for Cascading Style Sheets, is the language we use to style a webpage. CSS describes how HTML elements are displayed on a website (how small, big, what color), And they save a lot of work as well. It can control the style of multiple web pages, all at once. CSS is used to define style for your webpage, and is a really good idea to use if you want your webpage to stand out.

JavaScript

JavaScript is a scripting/programming language, that allows you to add more complex elements to your webpage. With JavaScript, you can add mor things to your webpage, like both 3d and 2d elements, interactive maps, or music players. It's a little more complex than HTML, or CSS, but the results that you can get out of JavaScript is amazing. JavaScript also stores API in its database, which are "are ready-made sets of code building blocks that allow a developer to implement programs that would otherwise be hard or impossible to implement".

AFrame

AFrame is a web framework that utilizes an entity-component (ECS) architecure, used for building 3D, VR, and AR experiences, using a mix of both HTML and JavaScript. It allows users to create immersive environments, without needing to know things such as WebGl or GLSL. It is also built on top of three.js and provides a simple, declarative way to create immersive 3D scenes.
